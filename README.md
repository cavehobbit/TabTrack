Tracks browser tab activity and organizes the results that show how time is spent across tabs.
It also gives you reminders when you get distracted for too long.
It also has a manual mode for adding websited on you own.
To track time automatically in real time, follow the following instructions:
Clone the repo,
Load the extension in Chrome using chrome://extensions.
Turn on dev mode & select the project folder 
Use the extension normally and get your daily reports!

Stores data via LocalStorage
made with Html, Css, Js & chrome extension APIs
